# Lamplight
Ever been practicing for an important test, say the SAT, and:
- your calculator ends up breaking/getting lost
- using a physical calculator ends up being inconvenient
- you need digital calculators for your tests that mimic ones you'd actually use in the test
- you need a centralised resource for all the calculators of all your tests
or something else?
You're in the right place!

## What is Lamplight?
A centralised, lightweight, easy-to-access calculator practice resource for exam/test prep. Tell it your name and what you're studying for, and it shows you exactly which calculators are approved for that board, then hands you a realistic working practice version of the actual model you'll use on test day.

## Running it
There's not really a build step. If you would like to use a local server, any static server works:
`npx serve .`

## Calculators
Six practice engines cover the ten boards/tests:
- **Casio ClassWiz** (fx-83GTX / fx-991EX family)
- **TI MultiView** (TI-30XS family)
- **TI-84 Plus CE / Casio fx-9750 / fx-CG50**
- **Desmos-style**
- **ETS on-screen calculator**
- **GMAT Focus on-screen calculator**

Calculator policies were checked against each board's current published rules (as of writing this in July 2026).
